Planificación de Rutas

Ionic Framework Enero - Febrero 2017 Aplicación Móvil

Aplicación realizada para la planificación de rutas de una empresa de transporte de valores. En la cual se planifica la visita a los clientes de la empresa para recoger el dinero, realiza la geolocalización del cliente. Genera reportes de entregas / recogidas. La base de datos de esta aplicacion esta desarrollada con Firebase, una base de datos en tiempo real, versatil y perfecta para construir aplicaciones en corto tiempo con buenas prestaciones y escalabilidad.
